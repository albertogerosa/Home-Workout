# Home-Workout

Gerosa Alberto – Progetto Web App
Titolo: Home Workout
Tagline

Palestra virtuale senza l'utilizzo di attrezzi costosi.

Descrizione

Home Workout è una web app pensata per aiutare le persone a mantenersi in forma direttamente da casa.
L’app è ideale per chi ha poco tempo per andare in palestra e per chi non vuole o non può acquistare attrezzature costose.
Propone allenamenti brevi, guidati, senza attrezzi e adatti a tutti i livelli.

Target

Persone con poco tempo a disposizione per allenarsi fuori casa.

Utenti con disponibilità economica limitata per attrezzatura o abbonamenti in palestra.

Principianti o utenti che vogliono allenarsi in modo semplice e immediato.

Competitors

Altre applicazioni dedicate all’allenamento fisico e all’home fitness.

Tecnologie Utilizzate

Frontend: HTML, CSS, JavaScript

Backend: PHP

Database: SQL

Link Pubblico alla Web App

https://daily-drill-buddy.lovable.app



Requisiti 
-come persona mi devo registrare al sito
-a seguito di quesiti per ogni persona voglio fornire un piano personalizzato per ogni persona
-come persona ottengo esercizi giornalieri 
-come persona voglio visualizzare i miei progressi giornalieri
-voglio inserire un  aumento di difficoltà col passare del tempo per ogni persona
-come persona visualizzo le statistiche per ogni esercizio
-come utente scelgo un orario giornaliero per allenarmi
-voglio inviare una notifica all'utente giornalmente all'orario impostato per l'allenamento
-come utente lascio un feedback all'app dopo ogni allenamento
-voglio fornire un periodo di riposo in base alla continuità della persona che svolge gli allenamenti
-voglio inserire la possibilità di fare amicizie tra utenti e fornire una classifica amici e una mondiale
-come utente posso visualizzare l'andamento dei miei amici
-come utente visualizzo i miei progressi finali terminato il piano d'allenamento
-voglio fornire la possibilità di creare un altro piano al termine di uno o salutare l'utente chiedendo un feedback finale sull'app
